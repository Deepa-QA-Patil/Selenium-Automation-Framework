# Real Auto Practice - Selenium Automation Framework

## Project Overview
This project is a Selenium-based Automation Testing Framework built using Java and TestNG.  
The framework follows industry best practices like Page Object Model, Data Driven Testing, Parallel Execution, and detailed reporting.

The goal of this project is to design a scalable and maintainable automation framework for web applications.

---

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- Git & GitHub
- Extent Reports

---

## Framework Features

✔ Page Object Model (POM) Design Pattern  
✔ Data Driven Testing using TestNG DataProvider  
✔ Parallel Test Execution using ThreadLocal  
✔ Cross Browser Testing  
✔ Extent Report Integration  
✔ TestNG Listeners Implementation   
✔ Action Driver Utility Class  
✔ Properties File Configuration  
✔ Structured Framework Folder Design

## How to Run the Project

1. Clone the repository
2. Open the project in Eclipse or IntelliJ
3. Install Maven dependencies
4. Run the `testng.xml` file

---

## Reports

After execution, Extent Reports will be generated showing:

- Passed Test Cases
- Failed Test Cases
- Screenshots on sucess
- Execution Logs
